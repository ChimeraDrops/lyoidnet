Simply Roleplay → JustRP FiveM Developer Early career
Started as a developer on Simply Roleplay, one of the earlier FiveM communities. The server was eventually sold and rebranded as JustRP. Stayed on through the transition before moving on to lead work.
Insurgency Roleplay Lead Developer
Built the entire Insurgency RP FiveM server from the ground up as lead developer. Designed and implemented core gameplay systems, scripts, and server infrastructure on the QBus framework.
  • Architected the full server stack — jobs, economy, housing, vehicle systems, custom scripts
  • Server grew to a sustained 120-player queue lasting several months at peak
  • Led a development team and coordinated releases with server ownership
  • Server eventually closed following ownership changes — built it, ran it, watched it fly
🏆 Peak: 120-slot queue — months of sustained population
Grit & Glory Founder / Lead Developer → Inactive Advisor Active → Ongoing
After Insurgency RP, created Grit & Glory — a RedM (Red Dead Redemption 2) roleplay server. Built the server, established the community, and handed ongoing operations to trusted co-developer CrazeDogg. Still an inactive advisor on the project.
  • Built on RedM framework with custom Lua scripting throughout
  • Repo: github.com/ChimeraDrops/GritAndGlory
Cypress RP Owner / Lead Developer In Development
Currently owned and in development. Cypress RP is being built as a GTA6 roleplay server — positioned for launch when GTA6 releases on PC. The website and foundational server structure are already underway.
  • Website: github.com/ChimeraDrops/CypressRP_Website — JavaScript, GPL v3.0
  • Includes 8 original HTML5 minigames built for the server (see below)
